### Main Idea
- The tutorial addresses the lack of standardization in early stages of recommendation pipelines, specifically data processing and multimodal feature extraction, which hinders comparability and reproducibility.
- It aims to provide theoretical insights and hands-on experience with tools and practices for standardized data processing and multimodal feature extraction in recommender systems.
- The proposed solution involves using two Python libraries: DataRec for reproducible data management and Ducho for streamlined multimodal feature extraction.

### Technologies
- **DataRec**: A Python library designed for reproducible and interoperable data management in recommender systems. It facilitates data filtering, splitting, and topological analysis.
- **Ducho**: A unified framework for extracting audio, visual, and textual features, integrating popular backends like TensorFlow, PyTorch, and Transformers.
- **Elliot**: A framework for recommender systems reproducibility and evaluation, used in conjunction with Ducho to run complete multimodal recommendation pipelines.
- **Datasets**: The tutorial covers widely used recommendation and multimodal recommendation datasets.
- **Evaluation Metrics**: Implicitly, the tutorial uses common evaluation practices integrated within Elliot framework.
- **Experimental Setup**: The hands-on sessions are designed to demonstrate the practical application of DataRec and Ducho, likely utilizing Google Colab for accessibility.

### Results
- The tutorial equips participants with theoretical and practical skills for managing multimodal recommendation datasets.
- Participants gain the ability to use DataRec to process datasets in a standardized and reproducible manner.
- Participants learn how to use Ducho to extract and process multimodal features, enhancing traditional recommendation models.
- The integration of DataRec and Ducho allows for creating reproducible and interoperable recommendation pipelines.
- The tutorial provides access to presentation slides, code, datasets, and a video recording, fostering knowledge sharing and further exploration.
