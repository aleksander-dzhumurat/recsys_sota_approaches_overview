### Main Idea

- The paper addresses the vulnerability of Visual-Aware Recommender Systems (VARS) to adversarial attacks, where malicious actors manipulate item images to unfairly boost product exposure.
- Current attack methods are either costly and detectable (shilling attacks) or fail to effectively integrate user preferences (visual-only perturbations).
- The authors propose AUV-Fusion, a novel cross-modal adversarial attack framework that leverages high-order user preference modeling and cross-modal adversary generation to create stealthy and effective recommendation poisoning. It integrates user interactions to create visually plausible perturbations, addressing the need for fake user profiles.

### Technologies

- **High-order User Preference Modeling:** This module integrates collaborative interactions and visual features using a multi-view strategy. It employs LightGCN to propagate multi-hop interactions on the user-item graph for robust behavioral embeddings and constructs an affinity graph based on the cosine similarity of raw image features for enhanced visual signals. The collaborative and visual representations are fused to create a comprehensive user embedding.
- **Cross-modal Adversary Generation:** This module incorporates user-driven adversarial adjustments into the latent space of a pre-trained diffusion model. User embeddings are processed through a multilayer perceptron (MLP) to produce a latent-space perturbation, which is injected into the VAE-derived latent representation during the forward diffusion process. Deterministic DDIM sampling is then used in the reverse process to reconstruct an adversarial image.
- **Hybrid Loss Function:** The framework employs a hybrid loss function comprising CLIP-based semantic preservation, SSIM for maintaining structural integrity, and a user alignment loss to balance attack potency and imperceptibility.
- **Datasets:** Amazon Men's Clothing dataset and Tradesy.com dataset are used for evaluation.
- **Evaluation Metrics:** Hit Rate (HR@k) is used to measure the exposure of target items. User preference scores are calculated to analyze overall imperceptibility.
- **Baselines:** INSA, EXPA, and SPAF are used as baselines for comparison.
- **Recommender Systems:** VBPR, DVBPR, and AMR are used as the target recommendation systems.
- **Graph Convolutional Networks:** Utilized within the High-order User Preference Modeling module to refine and enhance the semantic representations across connected items.
- **Diffusion Model:** A diffusion model with 50 steps is integrated to generate adversarial perturbations. Noise amplitude is adaptively regulated via a standard deviation parameter.

### Results

- AUV-Fusion significantly enhances the exposure of cold-start target items compared to conventional baseline methods across diverse VARS architectures and real-world datasets.
- In the Amazon Men setting for VBPR, the HR@5 obtained by AUV-Fusion is significantly higher than those of both EXPA and INSA. AUV-Fusion delivers notable improvements on AMR, indicating that the integration of high-order user preference data enables the generation of adversarial images that remain visually coherent and closely aligned with genuine user behavior.
- AUV-Fusion maintains exceptional stealth under rigorous scrutiny, as demonstrated by human user studies and semantic heatmap analysis. Semantic heatmaps of the images generated by the proposed model had more overlap with the heatmaps of the originals, compared to the baselines, meaning the adversarial images are more visually similar to the original images.
- Ablation studies confirm the importance of both the user alignment loss and the adversarial injection component in ensuring robust and reliable adversarial image generation.
- Experiments show that 50 diffusion steps are optimal for maximizing adversarial exposure gain while ensuring stability in reconstruction.
